Shimane: YNUPC for Android

Androidアプリのベータ版をAPKファイルとして公開します。

インストール方法

  1. 「設定」→「アプリケーション」→「提供元不明アプリ」にチェック入れます。
  2. こちらをクリックして、APKファイルをダウンロードします。
  3. (ここはしなくても結構です。)「設定」→「アプリケーション」→「提供元不明アプリ」のチェックを外します。
  4. ダウンロードが完了したら、通知バーに表示されるので、APKファイルをクリックしてインストールします。
  5. 「インストール」→「開く」
We released a beta version of YNUPC for Android to the public.

How to install

  1. "Settings"→"Apps"→Check the "Unknown Source" box.
  2. Click here in order to download the APK file.
  3. (You can skip this step.) "Settings"→"Apps"→Uncheck the "Unknown Source" box.
  4. Click the downloaded APK file on the status bar.
  5. "Install"→"Open"
